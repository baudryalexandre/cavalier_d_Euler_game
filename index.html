Jeu du Cavalier

Règles du jeu :
1. Le but du jeu est de déplacer le cavalier sur toutes les cases de l'échiquier sans repasser sur une case déjà visitée.
2. Cliquez sur une case pour déplacer le cavalier. Il peut se déplacer en forme de "L".
3. Vous gagnez lorsque toutes les cases ont été visitées !
4. Choisissez votre mode de jeu : solo (votre score est privé) ou compétition (vos scores seront affichés avec ceux des autres joueurs).

Let's go les tryharder, montrez votre skill !!!
